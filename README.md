# Flask-Trello-Backend

Este es el backend para una aplicación tipo Trello desarrollada con Flask.

## Requisitos

- Python 3.x
- pip
- pyenv (para gestionar el entorno virtual)
- Linux (o cualquier sistema operativo compatible)
